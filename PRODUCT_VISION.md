# My Assistant — رؤية المنتج وخارطة الطريق

> تطبيق موبايل يشتغل كـ"مدير شخصي": مش مجرد تراكر بيسجّل، لكن مساعد بياخد
> قرارات ويقترح ويلخّص. صحة + مواعيد + فلوس + مستندات + عادات في مكان واحد.

**المبدأ الحاكم:** أي تسجيل لازم ياخد أقل من ٣ ثواني أو يتسجل لوحده —
إدخال البيانات التقيل هو اللي بيموّت التطبيقات دي.

---

## الإصدار الأول (MVP) — اللي بنبنيه دلوقتي

| الوحدة | المحتوى |
|---|---|
| شاشة اليوم | تحية + ملخص "المدير" المبني على قواعد + مياه + نوم + مواعيد اليوم + جرعات اليوم + عادات اليوم + مصروف اليوم + تنبيه مستندات |
| المواعيد | CRUD كامل + فئات (شخصي/شغل/صحة/عيلة) + إشعار قبل الموعد بمدة قابلة للضبط |
| الأدوية | أدوية بأوقات جرعات متعددة + إشعار يومي متكرر لكل جرعة + تسجيل "اتاخدت" |
| المياه والنوم | عداد كوبايات بهدف يومي + تسجيل ساعات نوم بضغطة |
| الفلوس | تسجيل مصروف سريع (bottom sheet) + فئات + ميزانية شهرية + تقسيم بالفئة + تنقل بين الشهور |
| خزنة المستندات | صورة (كاميرا/معرض) + تاريخ انتهاء + إشعار قبل الانتهاء بمدة قابلة للضبط + شارات ملونة |
| العادات | سلاسل إنجاز مع "يوم رحمة" كل أسبوع + شريط آخر ٧ أيام |
| الإعدادات | الاسم + هدف المياه + ميزانية الشهر |

## الإصدار التاني

- **ذكاء حقيقي عبر `Claude API`**: محادثة بالعربي ("مش هلحق أتمرن النهارده" → إعادة جدولة)،
  تصوير روشتة → جدول أدوية تلقائي، تصوير فاتورة → مصروف متسجل
- **الأكل**: تسجيل وجبات سريع + سعرات تقريبية + خطة أسبوعية + قائمة تسوق
- **التمرين**: خطة أسبوعية + إعادة جدولة تلقائية لو فات يوم
- **الصلاة والورد**: مواعيد الصلاة بإشعارات + أذكار كعادات جاهزة
- **تكامل تقويم Google** (الخبرة موجودة من مشروع سابق)
- **`Health Connect` / `HealthKit`**: خطوات ونوم تلقائي من غير إدخال يدوي
- **قفل بالبصمة** (`local_auth`) — البيانات حساسة: صحة + فلوس + مستندات
- **`Home Widget`**: جدول اليوم + عداد المياه من غير فتح التطبيق
- **التسجيل بالصوت بالعربي**: "صرفت ١٥٠ بنزين وشربت ٣ مياه" → يتسجل لوحده
- **كارت الطوارئ**: فصيلة دم + حساسيات + رقم طوارئ من غير فك قفل

## المستقبل البعيد

- أهداف السنة + عجلة توازن الحياة (صحة/فلوس/علاقات/شغل/دين)
- طقس التخطيط الأسبوعي الموجَّه (١٠ دقايق كل جمعة)
- تقرير "سنتك" PDF + ذكريات ("في يوم زي ده من سنة")
- وضع العيلة: تطعيمات أطفال + قائمة تسوق مشتركة متزامنة
- ملف طبي بمنحنيات (ضغط/سكر بمرور الوقت) يتعرض للدكتور
- وضع المرض (تجميد العادات + كورس مضاد حيوي كل ٨ ساعات)
- أوضاع موسمية: رمضان (مياه من الفطار للسحور + إمساكية)
- كاشف التسويف ("المهمة دي اتأجلت ٣ مرات — نقسمها؟")
- العلاقات والمناسبات ("بقالك أسبوعين متكلمتش مع والدتك")
- تتبع القراية والكورسات
- منبه مربوط بأول موعد + وضع المساء (ملخص بكرة قبل النوم)
- شخصية المدير قابلة للضبط (حازم/لطيف)
- مزامنة سحابية اختيارية (Firebase) — التطبيق محلي أولًا دايمًا

> **المقايسة التفصيلية** (قيمة/مجهود/تكلفة لكل ميزة + ترتيب التنفيذ):
> راجع `ROADMAP_ESTIMATES.md`.

## قرارات منتج متفق عليها

1. **عربي فقط، RTL بالكامل**، أرقام عربية شرقية في العرض، وقبول الأرقام
   الشرقية في الإدخال.
2. **محلي أولًا (SQLite)** — بيانات حساسة، يشتغل من غير نت، والسحابة إضافة
   اختيارية مش أساس.
3. **الذكاء بالتدريج**: قواعد بسيطة الأول (نوم < ٦ ساعات → نصيحة)، و`Claude API`
   لما الأساس يثبت.
4. **يوم الرحمة في العادات**: يوم فايت واحد كل أسبوع مايكسرش السلسلة —
   كسر السلسلة أكبر سبب لهجر تطبيقات العادات.
5. أندرويد الأول؛ iOS لاحقًا عبر Codemagic (مفيش ماك متاح).
