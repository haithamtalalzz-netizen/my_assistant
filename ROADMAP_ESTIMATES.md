## 💡 بنك الأفكار (غير مجدولة — تتسحب منها الدفعات الجاية)

### جولة «أفكار مميزة» (2026-07-06) — كلها مجانية

**أتمتة حقيقية:**
- ⭐ قراءة رسائل البنك/فودافون كاش محليًا → تسجيل مصروف تلقائي (إذن SMS — اختياري وبشفافية)
- استيراد أعياد الميلاد من جهات الاتصال → مناسبات جاهزة
- تسجيل دخول صباحي تفاعلي (إشعار بأزرار: النوم/المزاج بضغطة)

**طابع خاص:**
- الأذان الصوتي الفعلي (صوت مخصص لقناة إشعارات الصلاة)
- وضع الجمعة (سورة الكهف + وقت الجمعة بدل الضهر + صلاة على النبي)
- مسبحة رقمية (عداد أذكار بضغطة + من الويدجت)

**ذكاء نفسي:**
- وضع «يوم صعب» (كتم غير الحرج + تأجيل + لهجة أخف)
- عداد الإقلاع «بقالك X يوم من غير...» + حساب الفلوس الموفرة
- «اليوم المثالي» (كل الأنظمة خضرا = يوم مثالي، عداد شهري)
- كبسولة زمنية (رسالة لنفسك تتفتح بعد سنة)

**تنظيم متقدم:**
- مشغّل الروتين (تسلسل موجه بمؤقت لروتين الصباح/المساء)
- شاشة «دلوقتي» (بس الحاجة الجاية — glanceable)
- قواعد تصنعها بنفسك («لو مصاريف الأسبوع عدت X نبهني») بقوالب
- تذكير الخروج من البيت (geofence — إذن موقع + بطارية بسيطة)

### جولة «الحياة المصرية اليومية» (2026-07-06) — كلها مجانية

**فلوس ومعاملات بلدنا:** (✅ الديون + الجمعية + صيانة البيت + الطقس اتنفذوا في الدفعة ٧ «بلدنا» 2026-07-07، v0.8)
- ✅ تتبع الديون والسلف («سلفت أحمد ٢٠٠» / «عليا لمحمد ٥٠٠») + تذكير لطيف
- ✅ متتبع الجمعية (القسط الشهري + دورك امتى + تذكيرات)
- قراءات العدادات (كهربا/مياه/غاز): تذكير شهري + سجل القراءات والفواتير
- متتبع المشاوير الحكومية (تجديد رخصة/باسبور: خطوات + ورق مطلوب + وصلت لفين) بقوالب جاهزة
- جدول تخفيف الأحمال (قطع الكهربا): تنبيه قبلها ١٥ دقيقة «اشحن موبايلك»

**صحة عملية:**
- صيدلية البيت (جرد أدوية البيت + صلاحيتها + «عندك بانادول متشتريش»)
- تقرير للدكتور (PDF بأدوية وقياسات ونوم آخر شهر — تطبعه للكشف)
- زر SOS في كارت الطوارئ (رسالة لجهة الطوارئ فيها موقعك الحالي)

**ذكريات ومتعة:**
- يوميات صوتية (دقيقة كل ليلة محليًا — بعد سنة أرشيف صوتي لحياتك)
- آلة الزمن (افتح أي تاريخ قديم وشوف يومه كامل من كل السجلات)
- أرقام قياسية شخصية (أطول سلسلة، أكتر خطوات، أقل شهر صرف)

**أوضاع وتخصيص:**
- وضع السفر/الإجازة (يوقف روتين الشغل + checklist شنطة + مواقيت صلاة تلقائية بالمكان)
- نغمة مميزة لكل نوع تذكير (تعرف الإشعار من صوته)
- ثيمات ألوان مخصصة
- وقت الشاشة كعادة عكسية (تقليل استخدام الموبايل — إذن usage stats)

### جولة «دين ومجتمع وبيت وشغل» (2026-07-06) — كلها مجانية

**دين وروابط:**
- ⭐ حفظ ومراجعة القرآن بالتكرار المتباعد (خطة حفظ + جدول تسميع تلقائي — المراجعة أهم من الحفظ)
- صلة الرحم الدورية (قايمة الأقارب + دورية لكل واحد «كل أسبوعين اتصل بعمك»)

**مجتمع مصري:**
- ⭐ سجل الواجبات الاجتماعية (النقوط في الأفراح + العزومات + العيديات — مين له إيه عندك ومردتش له)
- دفتر وصفات البيت (وصفة + صورة + مقادير + زرار «ضيف مقاديرها للتسوق»)

**البيت:**
- ✅ صيانة البيت الدورية (اتنفذت الدفعة ٧)
- ✅ أرشيف الضمانات (اتنفذت 2026-07-07، DB v17)
- ✅ صيدلية البيت (اتنفذت 2026-07-07، DB v16) — جرد + صلاحية + «عندك بانادول؟»
- ✅ قراءات العدادات (اتنفذت 2026-07-07، DB v18) — كهربا/مياه/غاز + استهلاك + تذكير شهري
- خزنة ملاحظات سرية ورا البصمة (أرقام حسابات، شفرات — مشفرة محليًا)

**شغل وفلوس:**
- «اتحرك دلوقتي» — تنبيه وقت التحرك للمشوار مش وقت الموعد (مدة المشوار بتتحدد مع الموعد)
- أهداف ادخار (عايز أجمع X → progress + «فاضل N شهر بمعدلك»)
- أظرف المرتب (توزيع تلقائي يوم القبض: التزامات/مصاريف/ادخار) + عد تنازلي ليوم القبض
- رصيد الإجازات السنوي (كام يوم فاضل + تسجيل اللي اتاخد)

**سياق خارجي (APIs مجانية بدون مفاتيح):**
- الطقس في ملخص الصبح عبر open-meteo («خماسين بكرة — اقفل الشبابيك» / «حر ٤٢ — زود مياه»)
- سعر الدولار في الملخص (اختياري — Frankfurter API مجاني)

**تحفيز:**
- تحديات شهرية جاهزة («شهر بلا سكر» — ٣٠ يوم بعداد وشارة إنجاز)

### جولة «عقل المدير الذكي» (2026-07-06) — توسعة تفصيلية للدفعة ٣ (Claude API)

**محادثة فاعلة (tool use — الرد بينفذ مش بيتكلم بس):**
- «مش هلحق أتمرن الأسبوع ده» → يعيد جدولة الخطة فعليًا
- «رتبلي يوم الجمعة» → خطة يوم من المعلق عندك (مواعيد فايتة + عادات + مشاوير)
- «اقفل الشهر» → جلسة مراجعة مالية تفاعلية
- أسئلة على بياناتك: «صرفت كام على الأكل بره آخر ٣ شهور؟»

**فهم الصور (vision):**
- روشتة → أدوية بجرعاتها ومواعيدها (المذكورة أصلًا)
- فاتورة سوبر → مصروف مفصل بالبنود
- ⭐ ورقة تحاليل → تخزين القيم في الملف الطبي + شرح مبسط (مع تحفظ: مش بديل للدكتور)
- صورة الأكلة → تقدير سعرات + تسجيل وجبة
- مستند حكومي → استخراج تاريخ الانتهاء تلقائي للخزنة
- منيو مطعم → «إيه أنسبلي حسب نظامي؟»

**تحليل عميق (مش قواعد جامدة):**
- الملخص الصباحي يتكتب بفهم السياق الكامل (يربط النوم بالجدول بالمزاج)
- التقرير الأسبوعي كنص إنساني حقيقي مش أرقام
- ⭐ اكتشاف ارتباطات خفية: «مصاريفك بتزيد الأيام اللي نومك فيها قليل»
- تفريغ اليوميات الصوتية + استخراج المواضيع المتكررة والمشاعر

**مستشارين متخصصين:**
- مدرب عادات (العادة اللي بتفشل باستمرار → يقترح تصغيرها أو نقل وقتها)
- مستشار مالي شهري (فين التسريب + ميزانية واقعية مبنية على أرقامك)
- مخطط وجبات أسبوعي حسب ذوقك وميزانيتك → قائمة تسوق تلقائية
- مساعد قرارات شراء: «أشتري دلوقتي ولا أستنى؟» من ميزانيتك وهدف ادخارك

**استباقية:**
- كشف التعارضات: «الموعد الجديد بيتعارض مع التمرين — أنقله؟»
- تحضير موعد الدكتور: ملخص أعراضك وأدويتك جاهز قبل الكشف
- «الأسبوع الجاي مزدحم — نخفف العادات مؤقتًا؟»
- سكرتير اجتماعي: صياغة رسائل (اعتذار عن عزومة، تهنئة مناسبة)

**قرارات تقنية متفق عليها للتنفيذ:**
- طبقة أفعال (tool use) بأدوات معرفة: add_expense / reschedule_workout / query_db...
- التكلفة: Haiku للتفاعلات السريعة + موديل أقوى للتقارير الأسبوعية فقط + prompt caching
- الخصوصية: البيانات تتبعت وقت السؤال بس (مفيش تخزين سحابي) + إعداد يتحكم في إيه اللي مسموح يتبعت
- fallback أوفلاين: من غير نت يرجع للقواعد المحلية الحالية بشفافية

## مؤجل عمدًا (قيمته لسه ماستحقتش مجهوده)

| الميزة | ليه مؤجلة |
|---|---|
| وضع العيلة + مزامنة سحابية | يحتاج Firebase + حسابات + أجهزة متعددة — قفزة معمارية كاملة |

## الخلاصة

- **الترتيب:** دفعة ١ → ٢ → ٣ → ٤. المجموع للوصول لـ«مدير فعلي» (نهاية دفعة ٣):
  **~٩–١٣ جلسة شغل**.
- **تكلفة التشغيل الكلية:** صفر جنيه لكل حاجة ما عدا الذكاء (~١–٥ دولار/شهر،
  اختياري ويتقفل).
- **القاعدة الحاكمة:** أي ميزة إدخالها يدوي وتقيل (الأكل تحديدًا) تستنى لما
  الصوت والصور يجهزوا — عشان كده الأكل في الدفعة ٤ مش قبلها.
