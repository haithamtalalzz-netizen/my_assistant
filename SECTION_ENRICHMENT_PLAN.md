# خطة تطوير الأقسام — My Assistant

> بدأت 2026-07-11. الهدف: نفس مستوى «الغِنى» اللي اتعمل في الرئيسية/الجدول/الفلوس/الصحة/
> العادات/المستندات/الوارد → نطبّقه على باقي الأقسام. المستخدم اختار «سيبها ليّا» =
> تنفيذ كل المراحل بالترتيب + build/install بعد كل مرحلة.

## نمط الإثراء الموحّد (يتطبّق على كل قسم)
1. **كارت ملخص/نظرة علوية** — عدّادات/إجماليات/تقدّم.
2. **قابلية قراءة بصرية** — أيقونات + ألوان للفئات، avatars.
3. **ترتيب بالأولوية** — المستحق/المنتهي/الأقرب الأول.
4. **مؤشرات اتجاه** — أسهم + دلتا مقارنة بالسابق.
5. **إجراءات تفاعلية** — toggle داخلي، سحب، اهتزاز.
6. **اتساق** — empty states + section headers موحّدة + `_attentionCard` عند اللزوم.

## المرحلة ١ — الفلوس/بلدنا  ✅
- [x] **الديون**: ترتيب بالأكبر مبلغ أول.
- [x] **الجمعية**: كارت ملخص «N جمعية مستحق قسطها + دورك الشهر ده» + ترتيب (دورك/مستحق أول).
- [x] **الادخار**: كارت «إجمالي مدّخرك» + نسبة كلية + عدد الأهداف اللي وصلت.
- [x] **صيانة البيت**: ترتيب المستحق أول + إصلاح اللون البني (amber tint بدل tertiaryContainer).
- [x] **صلة الرحم**: ترتيب المتأخر أول + عدّاد «محتاج تطمن على N» + إصلاح اللون البني.
- [x] **الواجبات الاجتماعية**: عنوان «مطلوب ترد لـ N» + إصلاح اللون البني في كارت الرصيد.

## المرحلة ٢ — الصحة واللياقة  ✅
- [x] **الجيم**: سطر إحصائي في كارت البرنامج «N تمارين الأسبوع • آخر تمرين من X يوم».
- [x] **التقدم البدني**: كارت علوي «الوزن الحالي + إجمالي التغيّر من البداية» (أخضر للنزول).
- [x] **الصيدلية**: ترتيب الأقرب انتهاءً + بانر «N منتهي • N قربت تنتهي» (خلال ٦٠ يوم).

## المرحلة ٣ — الأكل والمشتريات  ✅
- [x] **المشتريات** (`food/shopping_list_screen.dart`): ترتيب اللي ماتشالش أول + شريط تقدّم «N من M اتشالت» + اهتزاز. (الوجبات: حلقة السعرات موجودة أصلًا في الرئيسية+لوحة الصحة.)

## المرحلة ٤ — تطوير الذات  ✅
- [x] **قرآن**: بانر «N محتاج مراجعة النهاردة» + ترتيب المستحق أول + إصلاح اللون البني.
- [x] **التحديات**: (شريط التقدّم موجود أصلًا) + اهتزاز على «علّم النهارده».
- [x] **المذكرات**: هيدر «N يومية · آخر كتابة X» + تقصير المعاينة لسطرين.
- [x] **الإقلاع**: شريط تقدّم نحو أقرب محطة (أسبوع/شهر/٣/٦ شهور/سنة/سنتين) + «فاضل N يوم».

## المرحلة ٥ — شاشات النظرة العامة  ✅
- [x] **مركز التقارير/الرؤى/الرسوم/التقويم**: مكتملة أصلًا بالنمط المطلوب (كروت ملوّنة/ملخصات).
  **المراجعة الأسبوعية**: اتعملها إصلاح اللون البني ضمن م٦.

## المرحلة ٦ — تمرير اتساق شامل  ✅
- [x] **إزالة كل خلفيات `tertiaryContainer` البنية** عبر التطبيق → `scheme.tertiary.withValues(alpha:.13)`
  + أيقونة/نص `tertiary`: money (فاتورة + دخل دوري)، plants (بانر + مستحق)، weekly (المؤجَّلات).
  **صفر** استخدام لـ `tertiaryContainer` كخلفية كارت دلوقتي.

## المرحلة ٧ — أقسام البيت + الملف الطبي (تزويد أعمق)  ✅
- [x] **الضمانات** (`home/warranty_screen.dart`): ترتيب الأقرب انتهاءً + بانر «N قرب يخلص • N خلص» +
  أيقونة كهرمانية للقريب.
- [x] **العدادات** (`home/meters_screen.dart`): سهم اتجاه على الاستهلاك (أعلى=أحمر/أقل=أخضر) مقارنة
  بالفترة اللي قبلها.
- [x] **الملف الطبي / الملابس / الوصفات**: مراجَعة — بالفعل بمستوى الهدف (فلاتر/اقتراح outfit/شبكة).

## المرحلة ٨ — التقويم + الرسوم البيانية (تزويد أعمق)  ✅
- [x] **التقويم** (`calendar_screen.dart`): شريط ملخص «نشطت N يوم من M» + شريط تقدّم + زر «ارجع للشهر
  الحالي» في الـAppBar (يظهر لما تكون في شهر تاني).
